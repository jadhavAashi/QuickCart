# QuickCart
Worked on QuickCart – E-Commerce Application using Angular, ASP.NET Core, SQL Server, and Entity Framework Core. Developed responsive UI components, REST APIs, CRUD operations, authentication modules, and integrated frontend with backend services. Contributed to features like product management, shopping cart, order processing, and admin dashboard.

## Features
- User Authentication
- Product Management
- Shopping Cart
- Order Processing
- Admin Dashboard

## Tech Stack
- ASP.NET Core Web API
- Angular
- SQL Server
- Entity Framework Core
